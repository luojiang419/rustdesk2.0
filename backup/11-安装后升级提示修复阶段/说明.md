# 11-安装后升级提示修复阶段备份

## 备份范围
- `src/platform/windows.rs`
- `src/ui_interface.rs`
- `res/installer/windows/rustdesk2.0.iss`

## 问题原因
- Inno 安装包写入的是 `{7DB3360A-DC3B-44AA-9A46-9E41F6942B50}_is1` 卸载项，但运行时版本检测读取的是 `rustdesk2.0` 内置安装项。
- Inno 安装项没有 `BuildDate`，运行时将空值误判为“安装版本低于当前运行版本”。
- 点击“升级”会进入内置 `--install` 流程。该流程会先删除检测到的安装目录，再从当前目录复制文件；在 Inno 安装目录场景下可能把 Flutter 必需的 `data` 目录删掉或留下不完整安装，导致无法运行。

## 关键代码前后对比

### `src/platform/windows.rs`

修改前：

```rust
fn get_valid_subkey() -> String {
    let subkey = get_subkey(IS1, false);
    if !get_reg_of(&subkey, "InstallLocation").is_empty() {
        return subkey;
    }
    let subkey = get_subkey(IS1, true);
    if !get_reg_of(&subkey, "InstallLocation").is_empty() {
        return subkey;
    }
    let app_name = crate::get_app_name();
    ...
}
```

修改后：

```rust
fn get_valid_subkey() -> String {
    let app_name = crate::get_app_name();
    if app_name == "RustDesk" {
        let subkey = get_subkey(IS1, false);
        if !get_reg_of(&subkey, "InstallLocation").is_empty() {
            return subkey;
        }
        let subkey = get_subkey(IS1, true);
        if !get_reg_of(&subkey, "InstallLocation").is_empty() {
            return subkey;
        }
    }
    ...
}
```

### `src/ui_interface.rs`

修改前：

```rust
pub fn update_me(_path: String) {
    goto_install();
}
```

修改后：

```rust
pub fn update_me(_path: String) {
    if crate::common::is_custom_client() {
        log::info!("Skip built-in update flow for custom client");
        return;
    }
    goto_install();
}
```

修改前：

```rust
let b = crate::platform::windows::get_reg("BuildDate");
return crate::BUILD_DATE.cmp(&b).is_gt();
```

修改后：

```rust
if crate::common::is_custom_client() {
    return false;
}
let b = crate::platform::windows::get_reg("BuildDate");
return !b.is_empty() && crate::BUILD_DATE.cmp(&b).is_gt();
```

### `res/installer/windows/rustdesk2.0.iss`

新增：

```ini
#define MyRuntimeAppName "rustdesk2.0"
#define MyBuildDate GetDateTimeString("yyyy-mm-dd hh:nn", "", "")

[Registry]
Root: HKLM; Subkey: "Software\Microsoft\Windows\CurrentVersion\Uninstall\{#MyRuntimeAppName}"; ValueType: string; ValueName: "InstallLocation"; ValueData: "{app}"; Flags: uninsdeletevalue uninsdeletekeyifempty
Root: HKLM; Subkey: "Software\Microsoft\Windows\CurrentVersion\Uninstall\{#MyRuntimeAppName}"; ValueType: string; ValueName: "BuildDate"; ValueData: "{#MyBuildDate}"; Flags: uninsdeletevalue uninsdeletekeyifempty
```
