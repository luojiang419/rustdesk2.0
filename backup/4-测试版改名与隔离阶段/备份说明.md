# 备份说明 - 4-测试版改名与隔离阶段

## 创建时间
2026-06-30

## 阶段目标
将 Windows Flutter 测试版的软件名与主进程名调整为 `rustdesk2.0`，并沿用现有 `APP_NAME` 体系隔离安装目录、注册表、服务名、IPC 与配置目录，避免与本机已安装的 RustDesk 直接冲突。

## 备份范围
- `Cargo.toml`
- `libs/hbb_common/src/config.rs`
- `flutter/windows/CMakeLists.txt`
- `flutter/windows/runner/CMakeLists.txt`
- `flutter/windows/runner/Runner.rc`
- `flutter/windows/runner/main.cpp`
- `src/privacy_mode/win_topmost_window.rs`

## 修改前状态
- 运行时默认 `APP_NAME` 仍为 `RustDesk`。
- Windows Flutter runner 默认可执行名仍为 `rustdesk.exe`。
- Windows 可执行资源信息中的 `ProductName`、`FileDescription`、`OriginalFilename` 仍指向 `RustDesk` / `rustdesk.exe`。
- 依赖 `APP_NAME` 的安装目录、注册表、服务名、IPC 管道与配置目录尚未做测试版隔离。

## 预期改动
- 将默认 `APP_NAME` 改为 `rustdesk2.0`。
- 将 Windows Flutter runner 输出文件名改为 `rustdesk2.0.exe`。
- 同步更新 Windows 资源描述，保证任务管理器与文件属性显示为测试版名称。
- 在最小改动前提下保留现有构建链和多服务器中继逻辑不变。
