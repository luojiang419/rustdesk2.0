# 9-exe安装包构建阶段备份说明

## 阶段目标
- 生成 Windows x64 `.exe` 安装包。
- 安装包来源为当前已编译的 Flutter Release 目录：
  - `flutter/build/windows/x64/runner/Release`
- 输出目录：
  - `dist/`

## 修改范围
- 新增 Inno Setup 脚本：
  - `res/installer/windows/rustdesk2.0.iss`

## 代码前后对比

修改前：

```text
无独立 exe 安装器脚本，仅有便携 ZIP 包。
```

修改后：

```text
res/installer/windows/rustdesk2.0.iss
```

脚本负责：
- 将 `Release` 目录完整打包进安装器。
- 默认安装到 `Program Files/RustDesk2.0`。
- 创建开始菜单快捷方式。
- 可选创建桌面快捷方式。
- 安装后可选择启动 `rustdesk2.0.exe`。

## 下一步
- 使用 `ISCC.exe` 编译安装器脚本。
- 将产物放入 `dist/rustdesk2.0-windows-x64-setup.exe`。
