# 7-GitHub公开脱敏阶段备份说明

## 阶段目标
- 将 GitHub 主仓库和 `hbb_common` 子模块仓库调整为公开仓库。
- 公开前脱敏已跟踪的服务器配置文档，避免真实服务器地址和 key 进入公开历史。

## 修改范围
- `docs/远程服务器.md`

## 代码前后对比

修改前：

```text
<真实服务器地址 1>
key：<已脱敏>

<真实服务器地址 2>
key：<已脱敏>
```

修改后：

```text
# 远程服务器配置

公开仓库不保存真实服务器地址和 key。

本地测试时请在私有文档或本机配置中记录：

ID Server: <your-id-server>
Relay Server: <your-relay-server>
Key: <your-server-public-key>
```

## 备注
- 原始服务器信息仅存在于公开前的私有历史中，本阶段会重写主仓库公开分支和发布标签，避免公开引用保留原始值。
- 本备份文档故意不写入原始 key。
