## 具体修改的代码前后对比

### 1. `libs/scrap/Cargo.toml` bindgen 版本
修改前：
```toml
bindgen = "0.65"
```

修改后：
```toml
bindgen = "0.72"
```

### 2. `Cargo.lock` 中 scrap 依赖
修改前：
```text
"bindgen 0.65.1",
```

修改后：
```text
"bindgen 0.72.1",
```

### 3. `src/server_profiles.rs` tokio macro
修改前：
```rust
#[tokio::main(flavor = "current_thread")]
```

修改后：
```rust
use hbb_common::tokio;

#[hbb_common::tokio::main(flavor = "current_thread")]
```

### 4. `src/common.rs` no-default 音频 fallback
修改前：
```rust
#[cfg(feature = "use_dasp")]
pub fn audio_resample(...)

#[cfg(feature = "use_samplerate")]
pub fn audio_resample(...)
```

修改后新增：
```rust
#[cfg(not(any(feature = "use_dasp", feature = "use_samplerate")))]
pub fn audio_resample(
    data: &[f32],
    _sample_rate0: u32,
    _sample_rate: u32,
    _channels: u16,
) -> Vec<f32> {
    data.to_vec()
}
```

### 5. `src/server/audio_service.rs` 采样转换
修改前：
```rust
T: cpal::SizedSample + dasp::sample::ToSample<f32>,
let buffer: Vec<f32> = data.iter().map(|s| T::to_sample(*s)).collect();
```

修改后：
```rust
T: cpal::SizedSample,
f32: cpal::FromSample<T>,
let buffer: Vec<f32> = data.iter().map(|s| f32::from_sample(*s)).collect();
```

### 6. `src/server.rs` relay server 借用
修改前：
```rust
crate::check_port(relay_server, RELAY_PORT)
```

修改后：
```rust
crate::check_port(&relay_server, RELAY_PORT)
```
