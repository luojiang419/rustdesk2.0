## 具体修改的代码前后对比

### 1. `libs/hbb_common/src/config.rs`
修改前：
```rust
pub static ref APP_NAME: RwLock<String> = RwLock::new("RustDesk".to_owned());
```

修改后：
```rust
pub static ref APP_NAME: RwLock<String> = RwLock::new("rustdesk2.0".to_owned());
```

### 2. `flutter/windows/CMakeLists.txt`
修改前：
```cmake
set(BINARY_NAME "rustdesk")
```

修改后：
```cmake
set(BINARY_NAME "rustdesk2_0")
set(BINARY_OUTPUT_NAME "rustdesk2.0")
```

### 3. `flutter/windows/runner/CMakeLists.txt`
修改前：
```cmake
add_executable(${BINARY_NAME} WIN32 ...)
```

修改后：
```cmake
add_executable(${BINARY_NAME} WIN32 ...)
set_target_properties(${BINARY_NAME} PROPERTIES OUTPUT_NAME "${BINARY_OUTPUT_NAME}")
```

### 4. `flutter/windows/runner/Runner.rc`
修改前：
```text
VALUE "FileDescription", "RustDesk Remote Desktop"
VALUE "InternalName", "rustdesk"
VALUE "OriginalFilename", "rustdesk.exe"
VALUE "ProductName", "RustDesk"
```

修改后：
```text
VALUE "FileDescription", "rustdesk2.0 Remote Desktop"
VALUE "InternalName", "rustdesk2.0"
VALUE "OriginalFilename", "rustdesk2.0.exe"
VALUE "ProductName", "rustdesk2.0"
```

### 5. `src/privacy_mode/win_topmost_window.rs`
修改前：
```rust
pub const WIN_TOPMOST_INJECTED_PROCESS_EXE: &'static str = "RuntimeBroker_rustdesk.exe";
pub(super) const PRIVACY_WINDOW_CLASS: &'static str = "RustDeskPrivacyWindowClass";
pub(super) const PRIVACY_WINDOW_NAME: &'static str = "RustDeskPrivacyWindow";
```

修改后：
```rust
pub const WIN_TOPMOST_INJECTED_PROCESS_EXE: &'static str = "RuntimeBroker_rustdesk2.0.exe";
pub(super) const PRIVACY_WINDOW_CLASS: &'static str = "rustdesk2.0PrivacyWindowClass";
pub(super) const PRIVACY_WINDOW_NAME: &'static str = "rustdesk2.0PrivacyWindow";
```
